To run this project:

1. Navigate to project folder and run 'vagrant up' to set up the virtual environment.
2. 'vagrant ssh' to get into the virtual environment and navigate to the project folder
    using 'cd /vagrant'
3. Run 'python database_setup.py' and 'python fill_database.py' to set up the database.
4. Run 'python project.py' to start the server for the project.
5. Navigate to http://localhost:5000 in your browser to get to the application.
